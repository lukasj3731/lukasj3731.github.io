About

Description

Beetracker is a tool designed to measure activity of a camera-equipped hive. The camera streams a videofeed, which is analyzed frame by frame by a trained AI that is able to detect bees. Every bee is subsequently labeled and given a name associated with its tracking id. Bees that are detected to either enter or leave the hive are counted as activity, which is written hourly to a data file in addition to the current temperature. This data gives insight on the health of a hive, and also interesting correlations between temperature and activity.

Why

This project was a prototype for the "Innovation Management" department at T-Systems International. It was meant to augment the BeeAndme program and show how AI could be used to get even more useful data from a smart hive.
