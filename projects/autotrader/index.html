About

Description

Autotrader is meant to be an experiment to see if an AI can use twitter data to predict stock or cryptocurrency prices. It will use twitter trends or activity in order to predics price spikes and drops and trade accordingly, hopefully making a profit overall.

Why

In a fantasy world, this would be the last project we'd ever have to develop, since it would make enough money to survive off of it. In the real world, this is just an interesting showcase and a learning experience. Or at least I think so, who knows, this project might make millions when we end up starting development.
