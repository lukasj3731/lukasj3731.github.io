About

Description

Covid Forecast is a tool designed to predict how the infection numbres will develop within the next few weeks. The predictions are based on machine learning using Ludwig ML and are informed by current mobility data. The tool creates an optimistic, pessimistic and realistic guess in order to allow an assessment on lockdown measures or their effectiveness.

Why

This project was a prototype for the "Innovation Management" department at T-Systems International. It was created to see if AI could be helpful and accurate in assessing the development of infection numbers.

These values were extracted from an actual prediction of the AI. The AI is downloading fresh data in regular intervals, and uses the data to train a new model. With every new model, a new prediction is started, resulting in a newer and better projection of infection cases. Every cycle of new predictions contain 3 possibilities. An optimistic, a pessimistic and a realistic prediction. The optimistic prediction assumes that the mobility takes a low value, similar to a hard lockdown. The pessimistic prediction takes a mobility value close to a regular day before covid. The realistic values are based on the current mobility according to Googles mobility reports. Overall, the prediction returns data, that can be used to get a good approximation on where the case numbers are heading, and what effects a hard lockdown or removal thereof could have on it.
